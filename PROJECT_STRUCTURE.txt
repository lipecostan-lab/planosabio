🚀 COMO INICIAR
═══════════════════════════════════════════════════════════════

1. Abrir no navegador:
   → Arquivo → Abrir → src/index.html

2. Ou servir localmente:
   → python -m http.server 8000
   → Acessar: http://localhost:8000/src/

3. Adicionar novos arquivos:
   → HTML/CSS/JS → src/
   → Imagens → public/images/
   → Vídeos → public/videos/

4. Iniciar versionamento Git:
   → git init
   → git add .
   → git commit -m "Initial project setup"
